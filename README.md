# Example app with [`styled-components`][3] and [`hedron`][4]

## How to use

Download the example [or clone the repo][5]:

Install it and run:

```bash
npm install
npm run dev
```

Deploy it to the cloud with [now][6] ([download][7])

```bash
now
```
